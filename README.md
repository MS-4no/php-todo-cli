# php-todo-cli
PHP で作成したシンプルな TODO 管理の CLI アプリケーションです。  
タスクの追加・一覧表示・完了を CSV ファイルで管理します。

## 必要環境

- PHP 7.0 以上

## 使い方

### タスクを追加する

```bash
php todo_add.php "買い物に行く"
```

### 未完了のタスクを一覧表示する

```bash
php todo_list.php
```

出力例:

```
1:買い物に行く(2026-04-02 10:00:00)
2:PHPの勉強をする(2026-04-02 11:00:00)
```

### タスクを完了にする

```bash
php todo_finish.php 1
```

ID を指定してタスクを完了状態にします。

## ファイル構成

| ファイル | 説明 |
|---|---|
| `functions.php` | 共通関数・定数の定義 |
| `todo_add.php` | タスク追加 |
| `todo_list.php` | 未完了タスク一覧表示 |
| `todo_finish.php` | タスク完了 |
| `todo_list.csv` | タスクデータ (CSV) |

## バリデーション

- タスク名が未入力の場合はエラー
- タスク名が 140 文字を超える場合はエラー
- 完了時に不正な ID を指定した場合はエラー
